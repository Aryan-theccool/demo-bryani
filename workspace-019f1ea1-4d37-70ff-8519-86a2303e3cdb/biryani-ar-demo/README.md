# 3D Biryani AR Menu

This version uses a real GLB 3D model, not a flat moving image.

Files:
- index.html
- assets/biryani_plate_3d.glb
- assets/biryani-photo.jpg

Host the full folder on HTTPS, then send the live URL so the QR can be regenerated.

Android AR works with GLB via model-viewer Scene Viewer/WebXR. iPhone AR needs an additional USDZ export.
